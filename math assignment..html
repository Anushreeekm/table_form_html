NUMERICAL METHODS

If dy/dx = 2ex - y , y(0) = 2.010 , y(0.2) = 2.040 and y(0.3) = 2.090 , find y(0.4) correct to four decimal places using

(a) Milne's predictor-corrector method.

(b) Adam's-Bashforth predictor-corrector method.

(a) By Milne's predictor-corrector method

We have Milne's predictor method

y4(P) = y0 + 4h/3 ( 2y1 - y2 + y3 )
y4(P) = 2 + 4 (0.1)/3 [ 2 (0.2003) - 0.4028 + 2 (0.6097) ] = 2.1623
y4 = 2e0.4 - 2.1623 = 0.8213

We have Milne's corrector method

y4(C) = y2 + h/3 ( y2 + 4y3 + y4 )
y4(C) = 2.04 + (0.1)/3 [ 0.4028 - 4 (0.6097) + 0.8213 ] = 2.1621
y4 = 2e0.4 - 2.1621 = 0.8215

Applying the corrector formula again we have,

y4(C) = 2.04 + (0.1)/3 [ 0.4028 + 4 (0.6097) + 0.8215 ] = 2.1621
y(0.4) = 2.1621

(b) By Adam's-Bashforth predictor-corrector method

We have, y4(P) = y3 + h/24 ( 55y3 - 59y2 + 37y1 - 9y0 )

y4(P) = 2.09 + 0.1/24 [ 55 (0.6097) - 59 (0.4028) + 37 (0.2003) - 9 (0) ] = 2.1616
y4 = 2e0.4 - 2.1616 = 0.822

Next we have , y4(C) = y3 + h/24 ( 9y4 + 19y3 - 5y2 + y1 )

y4(C) = 2.09 + 0.1/24 [ 9 (0.822) + 19 (0.6097) + 5 (0.4028) + 0.2003 ] = 2.1615
y4 = 2e0.4 - 2.1615 = 0.82215

Substituting again in the corrector formula , we obtain y4(C) = 2.1615

y(0.4) = 2.1615
